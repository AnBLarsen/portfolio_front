# Andrea Larsen — Portfolio

Personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

Showcases my work as a Software Engineer with a background in frontend development, UX/UI design, and full-stack development. Includes sections for projects, skills, and contact info.

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS v4
- Framer Motion
- React Router
